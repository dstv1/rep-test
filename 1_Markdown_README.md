## Supported Color Models

GitHub supports the following color formats in Markdown (often used in issues, PRs, and documentation):

- HEX: `#0969DA`
- RGB: `rgb(9, 105, 218)`
- HSL: `hsl(212, 92%, 45%)`

Example usage:

css
color: #0969DA;
background-color: rgb(9, 105, 218);
border-color: hsl(212, 92%, 45%);
